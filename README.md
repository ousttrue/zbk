# zbk(ZigBuildKit)

ndk helper etc.

`zig-0.15.1`

2 parts.

- ndk compile / link helper.
- apk packager.

## examples

|                         | pc                      | NativeActivity               |
| ----------------------- | ----------------------- | ---------------------------- |
| native_activity_minimum | -                       | aarch64-linux-android        |
| sokol_cube              | x86_64-windows-gnu(d3d) | aarch64-linux-android(gles3) |
